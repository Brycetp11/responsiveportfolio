Covidocity

Covidocity is a page which allows the user to look at the current Covid-19 situation in a given Country or US State.

Pocket Pub Quiz

The Pocket Pub Quiz is a fun game which allows the user to create a unique quiz, with a chosen category. The user can then share the link with friends, and compare their scores to their friends. This app uses Sequelize, Express.js, Node.js, express-handlebars, and Materialize.

Music Quiz

This is a short quiz about Pop Music History, which tracks the user score. Be careful though, there is a time limit!

Daily Planner

The daily planner saves the user's input in each hour, and allows the user to examine their past, present, and future tasks to complete.
